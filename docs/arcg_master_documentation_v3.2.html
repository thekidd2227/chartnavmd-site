Non-Negotiable Rules

Fixed delivery windows and system constraints — no exceptions

Delivery Timeline — Fixed

Complex multi-system build30–45 days from signed scope

"90-day roadmap" and all references to a 90-day delivery cycle are permanently banned from all ARCG output.

1.2

No Deviation from the System Model

Every engagement uses Make.com + Airtable. Substitution requires owner sign-off and blueprint update before client work begins.

1.3

No Manual Workarounds

Fix the pipeline, not the symptom. Any manual intervention must be logged in the Airtable record's Notes field with reason, action, and root cause.

1.4

No Make UI Edits Without a Validated Blueprint

The Make UI generates invalid IML for raw JSON body fields. All blueprint changes go through the JSON blueprint editor only — validate externally before pasting.

1.5

Field Naming is Locked

Webhook payload field contactName must remain camelCase across form, webhook, and every Make module. Changes to Assessment.tsx require simultaneous Make module updates.
